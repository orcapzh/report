# 送货单合并工具

自动合并所有送货单的出货品类、数量和价格信息。

## 使用方法

1. 将所有送货单Excel文件放在 `raw-data` 目录下（支持子目录）

2. 运行脚本：
```bash
uv run python merge_delivery_orders.py
```

3. 查看生成的 `merged_delivery_orders.xlsx` 文件

## 输出说明

生成的Excel文件包含两个工作表：

### 汇总表
- 货名：产品名称
- 规格：产品规格
- 单位：计量单位
- 数量：累加后的总数量
- 平均单价：总金额/总数量
- 金额：累加后的总金额
- 客户：所有客户列表
- 文件：来源文件列表

### 详细数据表
保留每个送货单的原始记录，包含：
- 货名、规格、数量、单位、单价、金额
- 客户、日期、文件来源

## 环境要求

使用 uv 管理依赖，已安装：
- pandas
- openpyxl
- xlrd
